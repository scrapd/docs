# docs

The ScrAPD project documentation.

## Requirements

You need `python3` (with `pip`), `invoke` and `nox`.

```bash
pip3 install --user invoke nox
```

## Contribute

Set up the project locally:

```bash
inv
```

Serve it locally:

```bash
inv serve
```

Then go to <http://127.0.0.1:8000> to view the results.
